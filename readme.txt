== Description ==

All in One! Easily create a countdown, a current time, a current date and promoter as a sidebar widget or in a post or page using a shortcode. You can customize text, font color, background, font size and text align on your admin page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go and click the menu title ( Countdown, Date, Time and Promoter ) on admin page to configure the plugin.
4. Click the Save button after you customize your Countdown, Date, Time, Promoter and Design on the settings page.
5. Use the same format specified for your customized text on the settings page.
6. Use the short code [countdown_date_time_promoter] in a post or page or as a sidebar widget to display your plugin.

== Frequently Asked Questions ==

= Can I customize the align text, font size, background color and font color on the plugin settings page?  =

Yes, you can do all of your own design settings for your plugin.

= If I don't wanna display Countdown, do I have a chance to display only date or time on Wordpress page?  =

Yes, you have display options for Countdown, Date, Time, Promoter text and Promoter link.
